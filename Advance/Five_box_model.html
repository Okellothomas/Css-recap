The Box Model

Every element in a website is a box, including paragraph, headers among others!

The main components of box model include

  1. Main components
  2. Margin
  3. Padding
  4. Border
  5. Content

It is from inside out

The model is form inside out, that is, we start with the content, followed by the padding, then followed by the boarder, then finally the margin.
