Univeral Selector

The universal selector applied to the entire website and does not apply for the cases of inheritance

Element Selector

Apply to the elements of webpage as the a, p, and h1 tage and are least specific, they also apply to all the types of elements selected

Class Selector

Apply to all the elements of webpage are more specific compared to the element selector and apply to all the elements with the class selector name

Id Selector

Should only apply to a specific element in the entire website and its used should be minimized as possible. it is the most specific.

Group Selector

Apply to two more elements in the website with specific attribute of functionality.
